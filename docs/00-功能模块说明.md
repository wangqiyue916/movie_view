# 电影点评系统功能模块说明

## 1. 文档说明

本文档用于统一电影点评系统的功能模块划分和模块职责边界。系统采用“前台用户体验 + 后台内容运营 + 通用审核流程”的整体设计方式。

其中，电影信息、电影资讯、长评、解读视频、周边商品等内容均可进入统一投稿与审核流程。不同业务模块不重复设计审核状态，而是统一复用审核状态、审核记录、驳回原因和操作日志。

## 2. 通用审核流程

### 2.1 适用内容

通用审核流程适用于以下内容：

- 电影官方用户投稿的电影信息。
- 电影官方用户投稿的电影资讯。
- 电影官方用户投稿的电影解读视频链接。
- 电影官方用户投稿的电影周边商品。
- 普通用户发布的长评。
- 管理员需要审核、推荐、下架或处理的其他内容。

短评发布后默认直接生效，但被举报后可进入违规处理流程。

### 2.2 审核状态

系统统一使用以下状态：

| 状态 | 含义 |
| --- | --- |
| 草稿 | 投稿人保存内容但尚未提交审核。 |
| 待审核 | 投稿人已提交，等待管理员审核。 |
| 已通过 | 内容审核通过，但不一定已经正式展示。 |
| 已上线 | 内容已经对外展示。 |
| 已驳回 | 管理员审核不通过，并填写驳回原因。 |
| 已下架 | 已上线内容被管理员下架。 |
| 已删除 | 内容被逻辑删除，不再对外展示。 |

### 2.3 审核操作

管理员可执行以下操作：

- 通过：将内容由待审核变为已通过。
- 修改后通过：管理员小幅修改内容后通过审核。
- 驳回：填写驳回原因，内容变为已驳回。
- 上线：将已通过内容发布到前台。
- 下架：将已上线内容从前台隐藏。
- 删除：对违规或无效内容进行逻辑删除。

### 2.4 审核记录

每次审核操作都需要记录：

- 投稿内容类型。
- 投稿内容 ID。
- 投稿人 ID。
- 审核人 ID。
- 审核操作。
- 审核前状态。
- 审核后状态。
- 审核时间。
- 驳回原因或处理说明。

## 3. 功能模块划分

### 3.1 用户与角色管理模块

系统支持用户注册、登录、退出和账号状态管理。用户角色包括普通用户、管理员、超级管理员、电影官方用户。

普通用户可以搜索电影、浏览电影、评分、发布短评、发布长评、回复讨论、查看资讯、浏览商城、使用 AI 助手。

管理员负责电影信息、评论内容、长评内容、资讯内容、视频推荐、周边商品、首页推荐位等管理与审核。

超级管理员拥有最高权限，可以管理普通用户、管理员、电影官方用户及系统权限。

电影官方用户负责投稿电影信息、电影资讯、电影解读视频链接、电影周边商品，并查看投稿审核状态。

### 3.2 首页展示模块

首页顶部设置大面积滚动展示栏，用于展示当前热门电影资讯新闻。用户点击滚动栏中的资讯后，跳转到对应资讯详情页。

首页设置电影资讯栏目，用于展示最新资讯或热门资讯。首页还展示热门电影、高分电影、最新电影等电影预览信息。

首页电影预览中只展示电影总评分，不展示剧情评分、特效画面评分、演员演技评分等细分评分。

首页可展示优质高赞长评，该内容由长评模块统一提供数据，并可由管理员手动推荐或系统根据热度自动推荐。

### 3.3 电影搜索与浏览模块

用户可以通过电影名称、导演、演员、类型、上映时间等条件搜索电影。

系统展示符合条件的电影列表，用户可以点击电影进入电影详情页。

电影详情页展示电影基本信息、评分信息、短评、长评入口、解读视频、相关资讯和电影周边商品，帮助用户全面了解电影内容和用户评价。

### 3.4 电影信息展示与管理模块

系统展示电影名称、导演、主演、类型、上映时间、简介、海报、地区、语言、片长等基本信息。

管理员可以添加、修改、删除电影信息。电影官方用户可以投稿或补充电影信息，投稿内容进入统一审核流程，审核通过并上线后展示。

普通用户只能查看电影信息，不能修改电影数据。

电影详情页可以关联展示该电影的资讯、解读视频、长评和周边商品。

### 3.5 电影评分模块

普通用户登录后可以对电影进行评分。电影评分分为总评分和细分评分。

总评分为必评项，用户评分时必须填写。细分评分为选评项，包括剧情评分、特效画面评分、演员演技评分。

用户提交总评分后，评分结果立即生效。系统根据用户评分计算电影总平均分。

首页和电影列表页只展示电影总评分。电影详情页展示总评分、剧情评分、特效画面评分、演员演技评分。

每个用户对同一电影只能保留一条有效评分记录，用户再次评分时更新原评分。

### 3.6 短评模块

短评用于用户对电影进行简短评价。用户登录后可以在电影详情页发布短评。

短评发布后直接生效，并展示在电影详情页。短评适合用户快速表达观影感受。

用户可以对短评进行点赞、举报等互动。管理员可以删除违规短评或处理被举报内容。

短评违规处理不进入投稿审核流程，而是进入举报与内容处理流程。

### 3.7 长评与讨论模块

长评类似帖子，适合用户发布较完整、深入的电影评价。用户可以围绕剧情、人物、镜头语言、主题表达、导演风格、观影感受等内容撰写长评。

长评支持插入图片和 Markdown 格式展示，方便用户进行标题、加粗、引用、列表、图片等排版。

长评拥有独立详情页，用户点击后可以阅读全文。长评下方设置回复区，用户可以围绕长评内容进行讨论。

用户可以对长评进行点赞、收藏、回复、举报等操作。长评回复区默认支持一级回复，可根据系统复杂度扩展为多级回复。

普通用户提交长评后进入统一审核流程，审核通过并上线后对外展示。

优质长评可以根据点赞数、浏览量、回复数、收藏数等数据自动推荐，也可以由管理员手动推荐，并在首页展示。

### 3.8 AI 助手推荐模块

系统设置 AI 助手推荐功能。AI 助手以悬浮窗桌宠形式展示在页面中。

用户点击悬浮窗桌宠后，弹出 AI 对话框。用户可以向 AI 助手提问演员、导演、电影、电影周边、电影资讯等相关内容。

AI 助手根据用户问题返回对应介绍内容，也可以根据用户喜好推荐电影、长评、资讯和周边商品。

AI 推荐结果应支持跳转到电影详情页、长评详情页、资讯详情页或周边商品外链页。

### 3.9 电影解读视频推荐模块

电影详情页展示该电影相关的热门解读视频。视频内容可以包括剧情解析、人物分析、彩蛋解读、导演风格分析、结局解读等。

每个视频展示标题、封面、来源平台、简介或热度信息。用户点击视频后跳转到对应视频网站观看。

管理员可以管理视频推荐内容。电影官方用户可以投稿视频链接，投稿内容进入统一审核流程，审核通过并上线后展示。

### 3.10 电影资讯模块

系统提供电影资讯浏览功能。资讯内容包括新片上映、电影动态、演员新闻、导演动态、幕后花絮、获奖信息、票房新闻等。

首页顶部热门资讯滚动展示栏展示当前最热电影资讯新闻。首页电影资讯栏目展示最新资讯或热门资讯。

用户点击资讯标题、封面或滚动展示内容后，可以进入资讯文章详情页。

资讯文章可以关联具体电影、演员或导演。电影详情页可以展示与该电影相关的资讯内容。

电影官方用户可以投稿电影资讯，投稿内容进入统一审核流程。管理员负责资讯审核、发布、修改、删除和下架。

### 3.11 电影周边商城模块

系统设置独立的电影周边商城首页模块。商城主要用于展示电影相关周边商品的信息和图片，如手办、海报、服饰、徽章、文创产品、收藏品等。

用户可以在商城模块中浏览周边商品列表，查看商品名称、商品图片、所属电影、商品类型、价格、商品简介等信息。

系统不直接负责商品交易、订单和库存管理。用户点击感兴趣的周边商品后，可跳转到淘宝、京东等第三方电商平台的商品购买页面。

周边商品的信息、图片和外部购买链接由电影官方用户提供并上传，提交后进入统一审核流程。管理员审核通过并上线后，商品信息才能展示。

商城支持按照电影名称、商品类型、价格区间等条件进行筛选。电影详情页也会展示与该电影相关的周边商品。

### 3.12 内容投稿与审核模块

内容投稿与审核模块是系统的横向通用模块，用于承载长评、电影信息、资讯、视频、周边商品等内容的投稿、审核、上线、驳回、下架和记录追踪。

普通用户提交的长评需要管理员审核后展示。电影官方用户投稿的电影信息、电影资讯、解读视频链接、周边商品等内容需要管理员审核。

管理员可以对投稿内容进行通过、驳回、修改后通过、删除、上线、下架等操作。

管理员驳回内容时需要填写驳回原因，投稿用户或电影官方用户可以查看原因并修改后重新提交。

管理员可以设置优质长评、热门资讯、推荐视频和推荐商品。

### 3.13 管理员管理模块

管理员管理模块主要面向管理员和超级管理员，用于实现系统后台管理、内容审核、首页推荐位配置、违规内容处理和系统内容维护。

管理员可以进入管理员后台，对系统内容进行统一管理。

管理员可以审核各类投稿内容，包括电影信息、电影资讯、电影解读视频链接、电影周边商品、用户长评等。

管理员可以查看审核记录，包括投稿人、投稿类型、审核时间、审核结果、审核人和驳回原因等信息。

管理员可以管理首页推荐位，包括热门资讯滚动栏、电影资讯栏目、热门电影、高分电影、最新电影、优质长评、推荐周边等内容。

管理员可以处理违规内容，包括违规短评、违规长评、违规回复等，并根据违规情况对用户进行警告、限制发言或封禁账号等处理。

超级管理员可以管理普通管理员账号，并为不同管理员分配不同管理权限。

### 3.14 电影商管理模块

电影商管理模块主要面向电影官方用户或电影商用户，用于实现电影商后台、认证申请、官方内容投稿、审核状态查看和数据查看。

电影商用户可以提交认证申请，认证申请内容包括公司名称、联系人、联系方式、营业执照或电影授权证明等信息。

管理员或超级管理员审核通过后，电影商用户获得官方投稿权限。

电影商用户可以进入电影商后台，统一管理自己提交的官方内容。

电影商可以投稿电影相关内容，包括电影信息、电影资讯、电影解读视频链接和电影周边商品信息。

电影商可以查看投稿内容的审核状态，包括草稿、待审核、已通过、已上线、已驳回、已下架等。对于被驳回的内容，电影商可以查看驳回原因，并修改后重新提交。

电影商只能管理与自己绑定或授权的电影相关内容，不能修改其他电影商的内容。

电影商可以查看自己投稿内容的基础数据，例如电影浏览量、资讯阅读量、视频点击量、周边商品点击量等。

### 3.15 权限管理模块

权限管理模块用于根据用户角色控制系统功能入口和操作范围，保证不同身份用户只能访问和操作自己权限范围内的功能。

系统根据用户身份区分普通用户、管理员、超级管理员和电影官方用户。

普通用户可以访问前台功能，包括搜索电影、浏览电影、评分、发布短评、提交长评、参与回复讨论、查看电影资讯、查看解读视频、浏览电影周边等。

电影官方用户可以访问电影商后台，进行认证申请、电影信息投稿、电影资讯投稿、解读视频链接投稿、电影周边商品投稿和投稿状态查看。

管理员可以访问管理员后台，进行内容审核、内容管理、首页推荐位管理、违规内容处理等操作。

超级管理员拥有最高权限，可以访问全部后台功能，并管理普通用户、管理员和电影官方用户账号。

权限管理应覆盖页面访问权限、接口调用权限和数据操作权限。当用户尝试访问无权限页面或接口时，系统应进行拦截并提示无权限操作。
