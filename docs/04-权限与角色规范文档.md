# 电影点评系统权限与角色规范文档

## 1. 文档目标

本文档用于统一系统角色、权限、页面入口和接口访问控制规则。系统权限控制应覆盖前端路由、页面按钮、后端接口和数据归属校验。

## 2. 角色定义

| 角色编码 | 角色名称 | 说明 |
| --- | --- | --- |
| GUEST | 游客 | 未登录用户，只能访问公开内容。 |
| USER | 普通用户 | 可使用前台互动功能。 |
| OFFICIAL | 电影官方用户 | 可申请认证并投稿官方内容。 |
| ADMIN | 管理员 | 可进入管理员后台处理内容和审核。 |
| SUPER_ADMIN | 超级管理员 | 拥有最高权限。 |

## 6. 数据归属权限

### 6.1 电影官方用户

电影官方用户只能查看、修改和重新提交自己创建的投稿内容。

如果系统启用电影授权绑定，电影官方用户只能投稿或管理绑定电影相关内容。

电影官方用户不能直接上线内容，必须经过管理员审核。

### 6.2 普通用户

普通用户只能修改自己的长评草稿或被驳回长评。

普通用户不能修改已上线长评，如需修改可设计为重新进入审核流程。

普通用户只能删除或隐藏自己的短评和回复，不能处理他人内容。

### 6.3 管理员

管理员可以管理平台内容，但不能执行超级管理员专属账号和权限操作。

如果后续要细分管理员权限，可以按权限编码控制不同后台菜单和按钮。

## 7. 后端权限校验规则

后端必须进行以下校验：

- 是否登录。
- Token 是否有效。
- 用户账号状态是否正常。
- 用户是否拥有访问接口所需角色或权限。
- 用户是否拥有目标数据的归属权限。
- 内容当前状态是否允许执行目标操作。

禁止只依赖前端隐藏按钮进行权限控制。

## 8. 前端权限控制规则

前端应根据当前用户角色和权限控制：

- 顶部导航入口。
- 管理员后台入口。
- 电影商后台入口。
- 页面路由访问。
- 操作按钮显示。
- 无权限提示页。

前端权限控制只用于提升体验，最终权限以后端校验为准。

## 9. 无权限处理

| 场景 | 前端处理 | 后端处理 |
| --- | --- | --- |
| 未登录访问登录后功能 | 跳转登录页 | 返回 401 |
| 普通用户访问后台 | 显示无权限页 | 返回 403 |
| 电影商访问他人投稿 | 提示无权操作 | 返回 403 |
| 管理员访问超级管理员功能 | 隐藏入口或提示无权 | 返回 403 |
| 被封禁用户发布内容 | 提示账号受限 | 返回 403 |

## 10. 状态与权限联动

| 内容状态 | 投稿人可操作 | 管理员可操作 |
| --- | --- | --- |
| DRAFT | 编辑、提交 | 查看 |
| PENDING | 查看 | 通过、驳回、修改后通过 |
| APPROVED | 查看 | 上线、下架、删除 |
| ONLINE | 查看 | 下架、删除、推荐 |
| REJECTED | 查看原因、修改、重新提交 | 查看 |
| OFFLINE | 查看 | 重新上线、删除 |
| DELETED | 不可操作 | 按后台权限查看 |
