## 三、发现的 Bug 及修复

| # | Bug | 根因 | 修复方式 |
|---|-----|------|---------|
| 1 | Maven 不在 PATH | `mvn` 未添加到系统 PATH | 定位到 `C:\Temp\apache-maven-3.9.6` 完整路径调用 |
| 2 | Lombok 编译失败 | pom.xml 缺少 Lombok 依赖 | 添加 `org.projectlombok:lombok` |
| 3 | `PageResult.of()` 编译错误 | `PageResult` 是 `record` 类型，无 `of()` 方法 | 改用 `new PageResult<>(list, page, size, total)` 构造器 |
| 4 | `created_at` 插入为 NULL | 实体类标注了 `@TableField(fill)` 但缺少 `MetaObjectHandler` | 创建 `MybatisPlusConfig` 实现自动填充 |
| 5 | `review_images` 查询失败 | `selectImageUrlsByReviewId` 无 XML/注解映射 | 添加 `@Select` 注解 |
| 6 | `auditor_id` NOT NULL 约束 | 创建审核记录时未设置 `auditorId` | 设定 `audit.setAuditorId(userId)` |
